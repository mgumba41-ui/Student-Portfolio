System

Loan Mangement System

a group project designed to help manage and monitor loan records efficiently. The system allows users to add borrower information, record loan details, track payments, update loan statuses, and view loan balances. It also features a dashboard that displays the total number of loans, ongoing loans, fully paid loans, and overdue accounts. Through this project, we applied programming and database management concepts to create a simple and organized solution for handling loan transactions and records.
